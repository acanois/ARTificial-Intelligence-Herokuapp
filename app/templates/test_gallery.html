Machine Learning Models

We've built Supervised Machine Learning Models to predict the time period in which a painting by a specific artist was created. For each of the artists, we have models based on the Decision Tree Classification Algorithm and a Random Forest Classifier. We used python's image processing libraries to generate metrics based on the available art images. These are metrics that indicate the complexity of an image, contours, luminance, contrast, mean_rgb colors, etc. You can view these in the Data Tables.

The models use these metrics as the features for training and prediction and determine the time period or style as it varies over a time period. These models were tweaked and saved to be used for classifying any of the artists' art into these predefined time periods/styles of the artists' work. To see more painting of the astists categorized by the time periods/styles, visit the Artist's Gallery.
